# UncleCode Demo Visual Identity

## Style Prompt

Swiss Pulse for a local coding tool: precise grid composition, strong typographic hierarchy, high-contrast interface fragments, and crisp operational motion. The video should feel like a product operator switching a serious terminal system into a calm, capable work mode.

## Colors

- Background: `#f3f4f1`
- Ink: `#141414`
- Muted ink: `#565b60`
- Electric command blue: `#0057ff`
- Verification green: `#00a86b`
- Alert coral: `#f04f3a`

## Typography

- Display and UI labels: `Montserrat`, fallback Arial, sans-serif
- Code and telemetry: `IBM Plex Mono`, fallback `Courier New`, monospace

## Motion Rules

- Grid-snapped entrances with short overshoot and hard settling.
- Scene changes use mechanical wipes, block covers, or brief overexposure.
- Ambient motion stays subtle: counter ticks, scanner lines, and slow grid shifts.

## What NOT To Do

- No purple-blue gradients, glowing orb backgrounds, or decorative AI-dashboard haze.
- No rounded SaaS card soup; only functional panels, terminal panes, and data strips.
- No jump cuts between scenes.
- No tiny web-sized copy; every claim must read at video scale.
